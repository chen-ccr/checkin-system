# H5页面优化计划

## 任务概述
1. 同步离线队列：点击后先展示队列数据，上传成功后清空
2. 柱状图X轴标签显示在下方，Y轴显示具体数字，修复"范围"重叠问题
3. 考勤汇总列表右侧显示"实到 x/x"

## 具体步骤

### 任务1：同步离线队列展示
**文件**：`h5/src/App.vue`

**当前问题**：点击"同步离线队列"按钮后，直接上传，没有先展示离线队列数据

**修改内容**：
1. 在data中添加 `offlineQueueDisplay` 状态（布尔值）控制离线队列展示
2. 修改 `syncOfflineQueue` 函数：
   - 先展示离线队列（设置 `offlineQueueDisplay = true`）
   - 延迟200ms后再执行实际上传
   - 上传成功后清空队列并隐藏展示
3. 在模板中添加离线队列展示区域

### 任务2：柱状图X轴和"范围"布局修复
**文件**：`h5/src/App.vue`

**当前问题**：
- X轴标签（部门名）在柱状图上方，应该在下方
- Y轴没有显示具体数字
- "范围:xxx"和柱状图重叠

**修改内容**：
1. `.chart-label` 样式：添加 `margin-top` 改为负值，让标签显示在X轴下方
2. 添加Y轴数字显示（在 `.chart-grid` 上添加刻度线）
3. `.hint`（范围提示）添加足够的上边距，避免与图表重叠
4. 调整 `.chart-list` 的 `margin-top` 负值，使柱状图向上偏移，标签自然落到下方

### 任务3：考勤汇总列表显示"实到 x/x"
**文件**：`h5/src/App.vue`

**当前问题**：列表右侧显示 `x/x` 不够清晰

**修改内容**：
在模板的列表项中，将：
```html
<span>{{ row.actual }}/{{ row.expected }}</span>
```
改为：
```html
<span>实到 {{ row.actual }}/{{ row.expected }}</span>
```

## 注意事项
- 修改样式时注意移动端适配
- 确保修改后不影响其他功能
- 测试柱状图在不同数据量下的显示效果
